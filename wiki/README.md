# wiki/ - Obsidian 知识库

这是一个 Obsidian Vault，用于存放小溪编译后的 wiki 知识库。

## 目录结构

```
wiki/
├── 00-General/     # 通用知识
├── 01-AI/        # AI 相关
│   ├── Claude/   # Claude 架构学习
│   └── Agent/     # Agent 设计
├── 02-Engineering/ # 工程实践
└── 03-Life/      # 生活记录
```

## 使用方式

1. 用 Obsidian 打开这个目录作为 Vault
2. LLM 会增量编译 raw/ 中的数据到这里
3. 所有 .md 文件都是 LLM 维护的 wiki
